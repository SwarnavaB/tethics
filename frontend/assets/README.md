## TETHICS Brand Assets

- `tethics-logo-mark.svg`: primary shield mark
- `tethics-logo-horizontal.svg`: default horizontal lockup
- `tethics-logo-dark.svg`: light-on-dark lockup
- `tethics-logo-monochrome.svg`: one-color lockup
- `tethics-app-icon.svg`: square application icon
- `tethics-favicon.svg`: favicon / compact app icon
- `tethics-og-card.svg`: social preview / Open Graph artwork
- `tethics-brand-sheet.html`: printable press-kit / brand sheet

PNG fallbacks:
- `tethics-app-icon.png`
- `tethics-favicon.png`
- `tethics-logo-horizontal.png`
- `tethics-og-card.png`

Use the horizontal lockup by default in product surfaces. Use the mark alone only where horizontal space is constrained.
